Klávesové skratky


V ponuke je množstvo klávesových skratiek väčšiny príkazov špecifických pre aplikáciu Bean.


Nasleduje zoznam niektorých štandardných klávesových skratiek pre úpravu textu v systéme OS X. Niektoré z nich nie sú veľmi intuitívne, obzvlášť pre používateľov so skúsenosťami z Windows PC.


⌃⌘D … zobraziť vyskakovací slovník (láskavosť OS X)

[esc] … tipy na doplnenie slova (láskavosť OS X)

⌘↑ … kurzor na začiatok dokumentu

⌘↓ … kurzor na koniec dokumentu / force completed layout

⌥← … kurzor na začiatok slova (vľavo)

⌥→ … kurzor na koniec slova (vpravo)

⌃↑ … zobraziť stranu hore

⌃↓ … zobraziť stranu dole

fn↑ … rolovať zobrazenie o stranu hore

fn↓ … rolovať zobrazenie o stranu dole

fn⌥↑ … kurzor o stranu hore

fn⌥↓ … kurzor o stranu dole

⇧↑ … vybrať riadok hore

⇧↓ … vybrať riadok dole

fn⇧↑ … vybrať stranu hore

fn⇧↓ … vybrať stranu dole

⇧⌘↑ … vybrať všetko po začiatok dokumentu

⇧⌘↓ … vybrať všetko po koniec dokumentu

⌥⌫ … odstrániť časť slova vľavo

fn⌥⌫ … odstrániť časť slova vpravo

⌥[ … vynútiť dvojitú priamu úvodzovku (pri zapnutých živých úvodzovkách)

⌥] … vynútiť jednoduchú priamu úvodzovku (pri zapnutých živých úvodzovkách)

⌃[return] … vložiť zalomenie riadku (len v Leopardovi)

⌥[medzerník] … vložiť nerozdeliteľnú medzeru

[rolovacie koliesko]⌥ … zvlášť pomalé rolovanie (len v Leopardovi)

[ctrl]t … vzájomne zameniť dva znaky na pozícii kurzora


Pre zoznamy:


⌥[tab] … vytvoriť nový zoznam

[tab] … odsadiť položku zoznamu

⇧[tab] … znížiť odsadenie položky zoznamu

⌃⌥[tab] … vložiť tabulátor do položky zoznam

⌥⌘→ … odsadiť položku zoznamu (len v aplikácii Bean)

⌥⌘← … znížiť odsadenie položky zoznamu (len v aplikácii Bean)

POZNÁMKA: [return] na prázdnom riadku zníži odsadenie novej položky zoznamu


Panel pre hľadanie a zámenu:


⌥[tab] … vložiť znak tabulátora do poľa pre hľadanie

⌥[return] … vložiť znak return do poľa pre hľadanie

[option] … spôsobí prepnutie príkazov Nahradiť všetko/Vybrať všetko (v celom dokumente) na Nahradiť všetko/Vybrať všetko (len vo výbere)


Panel farieb:


⇧⌥ … spôsobí nastavenie farby zvýraznenia (pozadia) vybraného textu namiesto farby textu (len v aplikácii Bean)


Popis symbolov:


⌃ kláves Control

⌘ kláves Command

⌥ kláves Option (Alt)

fn kláves Function

⇧ kláves Shift

⌫ kláves Delete
