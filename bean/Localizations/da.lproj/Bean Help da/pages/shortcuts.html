Tastatur genveje


Der er kombination af genvejstaster i menuerne til mange specifikke Bean kommandoer.


I tillæg hertil er her nogle standard OS X tekstredigerings tastatur genveje. Nogle af dem er ikke intuitive - især for folk, der kommer fra en Windows PC baggrund.


⌃⌘D  ...  Åbn ordbog (intern OS X)

[esc]  ...  Tips til fuldførelse af ord (intern OS X)

⌘↑  ...  Cursor til begyndelse af dokument

⌘↓  ...  Cursor til slutning af dokument / Tving fuldført layout

⌥←  ...  Cursor til venstre for ord

⌥→  ... Cursor til højre for ord

⌃↑  ...  Vis én side op

⌃↓  ...  Vis én side ned

fn↑  ...  Rul side op

fn↓  ...  Rul side ned

fn⌥↑  ...  Cursor side op

fn⌥↓  ...  Cursor side ned

⇧↑  ...  Vælg én linie op

⇧↓  ...  Vælg én linie ned

fn⇧↑  ...  Vælg side op

fn⇧↓  ...  Vælg side ned

⇧⌘↑  ...  Vælg til begyndelse af dokument

⇧⌘↓  ...  Vælg til slutning af dokument

⌥⌫  ...  Slet ord mod venstre (slet baglæns)

fn⌥⌫  ...  Slet ord mod højre (slet fremad)

⌥[  ...  Tving lige dobbelte anførselstegn (når 'Grafiske anførselstegn' er aktiv)

⌥]  ...  Tving lige enkelt anførselstegn ((når 'Grafiske anførselstegn' er aktiv)

⌃[return]  ...  Indsæt linieskift (kun Leopard)

⌥[space bar]  ...  Indsæt ikke-brydende mellemrum

[scroller knob]⌥  ...  Ekstra langsom rulning (kun Leopard)

[ctrl]t  ...  Transponer de to tegn nærmest ved indsætningspunkt


Til lister:


⌥[tab]  ...  Opretter ny liste

[tab]  ...  Indrykker listeemne

⇧[tab]  ...  Fjerner indryk på listeemne

⌃⌥[tab]  ...  Indsætter tab-stop i listeemne

⌥⌘→  ...  Indrykker listeemne (kun i Bean)

⌥⌘←  ...  Fjerner indryk til listeemne (kun i Bean)

NOTE: [Linieskift] på en tom linie fjerner indrykninger på et nyt listeemne


Søgepanel:


⌥[tab]  ...  Indsætter tabulatortegn i søgefelt

⌥[linieskift]  ...  Indsætter tegn for linieskift i søgefelt

[option (ALT)]  ...  Medfører Erstat alle/Vælg alle (i hele dokumentet) til at blive Erstat alle/Vælg alle (kun i det valgte)


Farvepanel:


⇧[option (ALT)]  ...  Sætter markeringsfarve (baggrundsfarve) på valgt tekst istedet for skriftfarve (kun i Bean)


Symbolforklaring:


⌃ control key = kontroltast = CRTL

⌘ command  key = kommandotast = COM

⌥ option (eller alt) = alternativtast = ALT

fn function key = funktionstast = FN

⇧ shift key = skiftenøgle = SHIFT

⌫ delete key = slettetast = DEL
