快捷键


这些是大多数 Bean 特定的命令快捷键组合。


另外，也有一些是标准的 OS X 文本编辑的键盘快捷键。它们中有些并不是很直观，尤其是对于有 Windows PC 背景的人来说不太习惯。


⌃⌘D  ...  弹出字典 (OS X 内建)

[esc]  ...  单词联想提示 (OS X 内建)

⌘↑  ...  光标转到文稿开头

⌘↓  ...  光标转到文稿尾部 / 强制完成版面

⌥←  ...  光标转到单词左边

⌥→  ...  光标转到单词右边

⌃↑  ...  向上显示页面

⌃↓  ...  向下显示页面

fn↑  ...  向上滚动显示页面

fn↓  ...  向下滚动显示页面

fn⌥↑  ...  光标向上一页

fn⌥↓  ...  光标向下一页

⇧↑  ...  选择上一行

⇧↓  ...  选择下一行

fn⇧↑  ...  选择上一页

fn⇧↓  ...  选择下一页

⇧⌘↑  ...  选择到文稿开头

⇧⌘↓  ...  选择到文稿尾部

⌥⌫  ...  擦除左侧单词

fn⌥⌫  ...  擦除右侧单词

⌥[  ...  强制双直引号 (当 '智能引号' 开启时)

⌥]  ...  强制单直引号 (当 '智能引号' 开启时)

⌃[回车键]  ...  插入换行符 (仅 Leopard)

⌥[空格键]  ...  插入不间断空格

[滚动柄]⌥  ...  缓慢滚动 (仅 Leopard)

[ctrl]t  ...  调换插入点最近的两个字符


用于列表：


⌥[tab]  ...  创建新列表

[tab]  ...  缩进列表项

⇧[tab]  ...  不缩进列表项

⌃⌥[tab]  ...  插入制表位到列表项

⌥⌘→  ...  缩进列表项 (仅在 Bean 中)

⌥⌘←  ...  不缩进列表项 (仅在 Bean 中)

注意: 在空白行回车不会对新列表项缩进


查找面板：


⌥[tab]  ...  插入制表符到搜索框

⌥[return]  ...  插入回车符到搜索框

[option]  ...  使全部替换/全选 (在整篇文稿中) 变为全部替换/全选 (仅在所选部分中)


颜色面板：


⇧[option]  ...  将设置所选文本的高亮颜色 (背景颜色) 以替代字体颜色 (仅在 Bean 中)


符号说明：


⌃ control 键

⌘ command  键

⌥ option (或 alt)

fn function 键

⇧ shift 键

⌫ delete 键
