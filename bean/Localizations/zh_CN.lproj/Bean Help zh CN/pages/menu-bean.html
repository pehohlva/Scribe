Bean 菜单

关于 Bean：显示应用程序版本号和制作者/联系信息。

检查更新：检查是否有新版本的 Bean 可在 www.bean-osx.com 下载。您的 Mac 必须连接到因特网才能正常更新。注意 Bean 不会在启动时自动检查新版本。

偏好设置…

注意: 除非特别指示, 偏好设置只影响新创建的文稿，所以更改偏好设置中的设置不会影响已经打开的文稿。要影响打开的文稿，可用菜单栏的命令。

通用

连续拼写检查：给可能拼错的单词加红色的点下划线。使用 Ctrl+点按显示修改建议的上下文菜单。

智能拷贝/粘贴：剪切和粘贴单词时连按以自动添加和删除周围多余的空格。

后台分页：在您输入时后台将版面视图分页，而不是一次全部显示。这样，在记忆插入点在文稿开头时，可使得文稿在版面视图下载入更快。后台分页在新的 (Intel) Mac 上工作得非常好，但是不推荐用于老式的 (PowerPC) Mac，因为这可能会使系统变慢。用键盘强制分页的组合键：Cmd+Page Down。

使用 “智能引号”：用弯引号自动替换直引号 (或用您在样式控制中选择的引号以适合您的语言)。 

提供者为：如果 Bean 的智能引号功能与您的语言 ‘语音键’ 冲突而添加了区分符号到字母，则可以使用 > OS X 方案 (注意：仅 OS X 10.5)。

样式：(立即生效) 更改 Bean 智能引号引用标记的语言样式到用户首选的样式。

文本光标

垂直居中光标：在您结束文稿时将闪烁的文本光标居中于窗口 (仅用于版面模式)。不推荐用于老式 mac (G3 等)。

光标形状和行为：(立即生效) 提供三种类型的文本光标选择，包括 1) '标准' 窄的闪烁光标，2) 宽的闪烁和半透明光标， 3) 窄的不闪烁光标。

颜色：(立即生效) 更改文本光标的颜色。激活备选显示颜色后，光标颜色会混合到备选的文本颜色和背景颜色。颜色方块下面的默认按钮会还原光标为黑色。

文稿

默认存储格式：多信息文本格式 (.rtf) 对无图像的格式化文本效果很好；如果您经常包括图像到文稿，可以使用带图形的多信息文本格式 (.rtfd)。

自动存储文稿在每 ... 分钟： 激活 Cocoa 的自动存储功能，每次新打开的文稿都会在指定的时间间隔自动存储。如果 Bean 意外退出，下次启动时它将打开自动存储的文稿。

新建文稿模板：选取 '通用' 用于典型的新文稿 (注意默认页面大小可以在 OS X 偏好设置 > 打印与传真 > 默认纸张大小处设置)。选取 '自定' 以使用模板文件作为新文稿。选择 '选取模板' 按钮指定模板文稿 (仅锁定的文件可选择！)。要锁定文件用作 Bean 的文稿，选择文件 > 制作模板。或是选择显示 > 显示简介... 然后选择 '锁定的文件/模板' 注记格。一个文件也可以在 Finder 中锁定：选择文件，选择文件 > 显示简介然后选择 '锁定' 注记格。

打印

打印页眉和页脚：(立即生效) 打印到纸张或存储为 .pdf 格式 (显示在版面模式) 时将会添加页眉和页脚。字体族和大小在偏好设置 > 字体中选择。注意，如果文稿边缘很窄则页眉和页脚会挤到一起。 

样式： (立即生效) 更改打印时页眉页脚的内容。注意标题和作者属性由用户在显示 > 显示属性...中输入。

然后，您可以打印 APA 样式的页眉

An Analysis of Methodologies 1

...或是 MLA 样式的页眉

Kirkpatrick 1

...甚至是 ‘Turabian’ 样式的页眉 (在作者属性后添加四个空格)。

Kirkpatrick     1

页面开头：(立即生效) 指定在哪个页面页眉和页脚首次出现 (例如允许不编号的标题页)。页码在页眉和页脚会相应调整。

显示

版面视图：显示文稿为打印页面的效果，与连续的流文本相对。如果您处理大型文稿 (数百页) 您可能要禁用此选项以便加快文稿载入。

备选显示颜色：文稿以备选的前景和背景颜色显示。例如蓝底白字。这样对缓解眼睛的疲劳有一定作用。页眉和页脚不会以此显示。非黑色字体颜色不会被替换。

边距参考线：显示线条以指示可编辑区域，即边距内区域。边距参考线不会被打印。

页面阴影：版面视图下给页面添加阴影特效。如果关闭这个选项，Bean 可以变得更快！

实时字数统计：在窗口底部状态栏显示实时更新的字数和字符统计 (版面视图下还可统计页数)。也可显示选定文本的字数和字符统计 (这时会以蓝色字显示)。

标尺：在文稿上方显示标尺。您可以直接拖动标尺的箭头指示以调整文本的缩进量和制表位。在标尺中点按以创建左侧制表位停留点。

标尺附件：在标尺上显示标准的 OS X 标尺控制。启用后，您可以拖动不同类型的制表位停留点。您还可以访问 OS X 原生的样式控制。

工具栏：工具栏图标可快速直观的访问常用命令。

水平滚动条：允许您在放大的文稿中水平滚动文稿。

不可见字符：显示其它不可见字符 (空格符，制表符，回车符，段落记号以及分页符) 以便于您控制文稿版面。启用后，可以方便的拷贝这些字符用于查找/替换面板 (查找和替换不可见字符)。

颜色：指定不可见字符显示时的颜色。

不显示空格：当启用了 '显示不可见字符' 时不显示空格符。可以减少一些干扰。

字体

多信息文本字体：用于新文稿的默认字体和大小。提供了预览。

纯文本字体：用于载入纯文本文稿 (.txt，.html，等) 时的默认字体和大小。提供了预览。

使用备选显示颜色：一旦启用，您可以点按文本和背景色来更改使用的替换显示颜色。

样式

边距：提供新文稿的边距设置。点按 ‘应用更改’ 按钮以存储更改。

行距：提供新文稿的行距设置。行距 1.2 的效果是比较好的。

首行缩进：当用户按下回车键时提供自动段落缩进设置。  点按 ‘应用更改’ 按钮以存储更改。

应用样式到 .txt 文件：应用样式面板的设置到 Bean 打开的纯文本文件。这不是始终必要的。例如对于编辑 HTML 代码，您就不会想进行格式化，因为可能会使代码变得不清晰。但对于印刷用的文本文件来说这些设置可使文件增强显示和阅读效果。

以公制单位显示：显示样式面板的控制设置，以公制单位取代美制单位。这不影响新文稿窗口的标尺等。(这是由系统偏好设置 > 多语言环境 > 格式 > 度量单位自动设置的)。

应用更改：这个按钮使您在此面板所作更改生效。

窗口

居中首个窗口：新文稿窗口在显示器居中，之后新建窗口会层叠排在之前窗口的右侧。

窗口大小

初始设置：新窗口以默认大小打开。

自定：新窗口根据用户检测的大小打开。

匹配到活动文稿：设置自定窗口大小以匹配最前面的的文稿窗口。

缩放

适合页宽：新文稿进行了缩放以便打印页面宽度适合文稿窗口。当用户调整了窗口大小，文稿会随之放大或缩小。

除了纯文本：对纯文本文件禁用 ‘适合页宽’。一般对纯文本，用户会经常调整窗口大小以避免自动换行，因此调整大小没必要。

自定义：新文稿按用户定义值来缩放。

全屏幕选项

全屏幕选项

左/右全屏幕边距：控制可编辑文本周的水平边白总量 (屏幕上的左和右边距)，显示为屏宽的百分比 (每边 33% 为最大)。

隐藏标尺：全屏幕下隐藏标尺并在退出全屏幕时恢复。

隐藏工具栏：全屏幕下隐藏工具栏并在退出全屏幕时恢复。

使用备选文本颜色：在全屏幕下启用备选文本颜色并在退出全屏幕时禁用。

隐藏版面视图：全屏幕下隐藏版面视图并在退出全屏幕时恢复。

高级 (文稿)

允许更改文稿的背景颜色：在字体面板显示 ‘颜色方块’ 控件以更改文稿的背景色。文稿的背景色将打印 (与显示 > 备选颜色的颜色不同)，但不会打印文稿的边距。

采用反锯齿阈值 / 关闭智能字距紧排：让 Bean 采用系统偏好设置的 '外观 > 关闭字体平滑在字体大小在 ... 以下时' (程序重启时生效)。当启用屏幕字体时可使用更少的系统资源，它也禁用每个大小的文本的所有智能字距紧排 (字符的间距)，所以不鼓励这样做。

'粘贴所选部分...' 服务添加分隔符：让服务 > Bean > 粘贴所选部分到当前 Bean 文稿以插入图形分隔符到插入所选文本之前。因此，当服务重复使用时，不同的文本会被划分开。(高级提示：分隔符字串自身存储于 defaults.plist 中，位于键 prefPasteSelectionDividerString 处，可以由用户自己去替换。)

默认分栏间的装订线 (以磅为单位)：设置在格式 > 分栏 > 装订线控制的量 (栏数为两个以上)。

高级 (界面)

度量单位：让 Bean 显示度量 (以标尺和控件) 使用公制单位或是美制单位。自动设置将让 Bean 根据系统偏好设置的多语言设置情况来使用公制或美制单位。

打开文稿时还原光标位置：让 Bean 在重新打开文稿时自动还原上次存储时光标所在位置 (仅用于多信息文本格式)。关闭此选项文稿可以载入更快。

在状态栏显示页码 (仅版面视图)：激活版面视图和实时字数统计时，在状态栏指示页面可见范围和总页码。

在版面视图显示页码：版面视图的页面会编号 (只是上面和到每页的左边)。

显示垂直标尺：顾名思义。(仅 OS X 10.5+)

首次存储时建议文件名：(立即生效) Bean 会在存储面板上根据文稿前 40 个字符建议文件名。(仅 OS X 10.5+)

高级 (查找/替换)

使用简化查找/替换面板：可以使查找 > 查找/替换... 命令显示标准的 OS X 文本查找面板，以取代 Bean 的 '高级' 查找面板。Bean 的查找面板允许使用匹配模式 (正则表达式)，而简化的查找面板不能使用。

^ 和 $ 识别新行：(立即生效) 会使正则表达式符号 ^ 和 $ 识别除文稿通常开头和结尾之外内嵌的新行。注意找到的文本不会包括新行，只是它之后 ( ^ ) 或之前 ( $ ) 的文本。

不匹配新行：(立即生效) 会使 '句点' 正则表达式符号 ( . ) 匹配新行，与通常其它字符一样 (除了新行)。找到的文本将包括新行。

高级 (备注模式)

设置字体和颜色以匹配所选文本：(立即生效) 设置备注模式所用的文本属性 (字体名称、字体大小、文本颜色和高亮颜色)。这些属性来自前端文稿光标处的文本。

使用字体/大小；使用文本颜色；使用高亮颜色：(立即生效) 选择您想在备注模式中应用的属性。

添加括号：(立即生效) 用方括号括起新的备注。

添加新段落：(立即生效) 用新的备注创建新段落。

服务

从所选部分到带样式的 HTML 片段：将所选文本以兼容程序转换为带内嵌 CSS 的 HTMl 片段，CSS 在其自己的 Bean 文稿窗口打开。HTML 片段 (文本，列表，表格等) 可以直接粘贴到 HTML 文稿中。

从所选部分到不带样式的 HTML 片段：将所选文本以兼容程序转换为不带带 CSS 的 HTMl 片段，CSS 在其自己的 Bean 文稿窗口打开。

建立包含选定内容的新文稿：创建新的包含所选文本的未命名文稿 (可以从网络浏览器等程序选择文本)。

粘贴 ‘Lorem Ipsum’：粘贴一段印刷测试用的拉丁语文本到任何文字编辑器。

粘贴所选到当前 Bean 文稿：从任何程序粘贴所选文本到前端打开的 Bean 文稿。要自动在每个文本前添加分隔符，激活此选项：Bean 偏好设置 > 高级 > 文稿 > '粘贴所选部分...' 服务添加分隔符。

从所选文件恢复文本：从损坏或传统格式的文件恢复纯文本内容。在 Finder 中选择一个文件，然后在服务菜单选择此服务。Bean 会创建一个第一部分包含恢复文本的新文稿 (格式，图像等不能恢复)，第二部分包括忽略的文本 (一般是无用数据)。控制符可能会使 Bean 在处理中崩溃。使用此服务可从损坏的 Word 97 格式恢复，或从较老的格式恢复。注意：Dataviz 出售的 MacLinkPlus 可以转换大多数当前或传统的文件格式。
