Résolution de problèmes...


Si je fais une modification dans les préférences, mes documents ouverts ne se modifient pas.

Les préférences concernent en général les nouveaux documents. Afin de modifier un document ouvert, servez-vous des éléments de la barre de menu.


Bean se plante au démarrage.

Essayez de mettre à la poubelle le fichier suivant (qui contient les préférences de Bean), puis redémarrez Bean:


/UTILISATEURS/<VOUS>/Library/Preferences/com.SevenYearsDreaming.Bean.plist


Si Bean se plante à plusieurs reprises lors du démarrage, cela peut s'expliquer par le fait que le logiciel essaie d'ouvrir un document auto-enregistré qui contient corrompu. Essayez de supprimer les documents auto-enregistrés dans le dossier suivant:


UTILISATEURS/<VOUS>/Library/Autosave Information/

Des puces de numérotation apparaissent de nulle part.

Supprimer la puce initiale ne suffit pas pour supprimer une liste: vous devez enlever l'attribut de liste sous-jacent. Vous pouvez le faire en appliquant Format > Liste > Marqueur… > Aucun à une sélection, ou en appliquant Édition > Supprimer > Marqueurs de liste sélectionnés à une sélection.


Des parties du document disparaissent en mode Mise en page.

La pagination est probablement déroutée par des tableaux de texte invisibles copiés et collés depuis une page web (un bogue connu). Essayez 

Édition > Supprimer > Tableaux. Afin de faire apparaitre les tableaux cachés, sélectionnez Affichage > Afficher les caractères invisibles.


L'icône «Installer Bean» ne disparait pas.

D'abord, quelques informations générales: La manière la plus habituelle d'installer des logiciels sur un Mac est probablement celle au moyen d'une image disque. Une image disque est un disque «virtuel» contenu dans un fichier portant le suffixe .dmg. Double-cliquez un fichier .dmg afin d'ouvrir le dossier contenant le disque virtuel (normalement un logiciel ou un installateur, parfois aussi de la documentation). Souvent, vous pouvez lancer le logiciel sur place, depuis le disque virtuel, afin de voir si vous l'appréciez. Pour un usage quotidien, glissez l'icône du logiciel depuis l'image disque dans votre dossier Applications afin d'y faire une copie.


Maintenant, un conseil spécifique: D'abord, si vous voulez garder une copie de Bean pour l'usage quotidien, assurez-vous que vous avez copié Bean dans votre dossier Applications. Non pas l'image disque Bean, mais bien le fichier Bean.app que vous apercevez si vous ouvrez Bean-Install.dmg en le double-cliquant. Puis, glissez l'image disque ouverte nommé Bean-Install vers l'icône de la corbeille dans le Dock: ceci «éjectera» l'image disque (comme vous éjecteriez une clé USB ou un DVD après vous en être servi). Vous pouvez ensuite supprimer le fichier nommé Bean-Install.dmg (qui se trouve probablement sur votre bureau ou dans le dossier Téléchargements). N'oubliez pas que Bean est gratuit, de sorte que vous pouvez des copies de l'image disque à d'autres si vous le désirez.


Un document ne s'ouvre pas avec Bean.

Normalement, Bean détermine le format d'un fichier sur la base de son suffixe (.doc, .rtf, .txt, etc.). Si le nom de fichier ne contient pas de suffixe, essayer d'en ajouter un.


Un document .doc est devenu illisible.

Je recommande d'utilise le format Rich Text (.rtf) comme format d'enregistrement par défaut, spécialement pour les longs documents, puisque c'est le format natif de Bean. Les formats Word (.doc) sont toujours importés et exportés, ce qui comporte une étape supplémentaire, où il peut y avoir des erreurs. Si vous vous trouvez face à un document corrompu ou un format de fichier obsolète, essayez le service Services > Bean > Rétablir le texte du fichier sélectionné.


Comment faire des notes de bas de page, des colonnes ou des sauts de section?

Vous aurez besoin d'un autre traitement de texte. Un qui est bon et gratuit est iText Express 3.0. Il y aussi OpenOffice Aqua (également gratuit).

Est-ce que je peux rédiger mon travail de diplôme avec Bean?

Pour certaines choses, vous avez besoin d'un véritable traitement de texte ou alors de TeX. Pour des choses comme un travail de diplôme.
