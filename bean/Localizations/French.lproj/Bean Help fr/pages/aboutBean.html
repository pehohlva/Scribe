À propos de Bean...


Bean: qu'est-ce que c'est?

Bean est un éditeur de texte petit, efficient et riche pour OS X. Il offre un compteur de mots en temps réel, une réglette de zoom, des sauvegardes munies d'une date, un mode "mise en page" et un mode "couleurs alternatives" pour le traitement de texte et un mode "plein écran". Il a un inspecteur pour ajuster le formatage du texte. Vous pouvez sélectionner le texte par le style, puis coller un autre style sur la sélection.


Bean fonctionne sous Tiger et Leopard, sur les Macs PowerPC et Intel. Il se base sur le dispositif de gestion de texte Apple Cocoa text engine (NSText) et était basé sur NSDocument même avant Text Edit.

Ce que Bean ne fera pas…

... les notes de bas de page, colonnes, sections ou fenêtres à affichage double. De même, il n'y a pas de couche graphique, seulement les images dans le texte.


Bean est gratuit et open source

Bean est gratuit; utilisez-le et partagez-le si vous le voulez. Bean est édité avec une licence Gnu GPL; cela veut dire que le code source est disponible et que vous pouvez l'utiliser ou l'améliorer. Notez que d'après la licence Gnu GPL license, si vous utilisez le code source de Bean pour des projets non privés, votre code doit également être en open source.

Si Bean se plante à cause d'un bogue, veuillez copier le message d'erreur depuis Console.app et me l'envoyer. Merci!

Avertissement:

N'utilisez pas Bean pour des travaux critiques. Vous voulez écrire une thèse de doctorat? Utilisez un autre logiciel. Si vous écrivez quelque chose d'important ou de conséquent, je vous conseille d'utiliser les fonctions de sauvegarde et d'enregistrement automatique. Et sauvegardez votre disque dur (p. ex. à l'aide de Time Machine). La question n'est pas de savoir si le disque va rendre l'âme, mais plutôt quand (ceci sera d'actualité tant que la mémoire permanente n'est pas stockée sur des supports flash.
