Menu Bean

À propos de Bean: affiche la version de Bean, quelques crédits et des informations pour le contact.

Chercher les mises à jour: vérifie s'il y a une mise à jour à télécharger sur www.bean-osx.com. Votre Mac doit être connecté à internet pour que cette fonction marche. Notez que Bean ne vérifie pas automatiquement s'il y a une nouvelle version à chaque ouverture.

Préférences…

NOTEZ: Si ce n'est pas indiqué différemment, les préférences affectent la création de nouveaux documents. Ainsi, si vous changez un réglage dans les préférences, cela ne modifiera pas le document ouvert. Utilisez les commandes dans la barre de menu afin de modifier un document ouvert.

Général

Vérification de l'orthographe lors de la frappe: souligne en rouge des mots qui sont peut-être mal écrits. Un menu contextuel (Ctrl-clic) permet de faire afficher les corrections suggérées.

Copier/coller intelligent: ajoute et supprime automatiquement les espaces avoisinants si vous coupez et collez des mots sélectionnés par un double clic.

Pagination en tâche de fond: repagine en mode Mise en page lors de la frappe au lieu de le faire en une seule fois. Ceci permet de charger les documents plus rapidement dans le mode Mise en page si le point d'insertion pressenti se trouve vers le début du document. La pagination en tâche de fond fonctionne bien sur les nouveaux Mac (Intel), mais elle n'est pas recommandée pour les machines (PowerPC) plus anciennes, car elle peut ralentir le système. Forcez la repagination avec le raccourci clavier Cmd+Page suivante.

Guillemets symétriques: remplace automatiquement les guillemets droits par des guillemets symétriques ouvrants et fermants (c'est-à-dire les guillemets correspondant à la langue de l'utilisateur, telle qu'elle est sélectionnée dans le panneau de contrôle Style).

Fourni par: si cela interfère avec l'utilisation de touches «muettes», employées pour les lettres diacritiques, sélectionnez Fourni par > OS X (notez: seulement dans OS X 10.5).

Style: (s'applique immédiatement) modifie le style des guillemets utilisé par Bean en fonction du style préféré de l'utilisateur.


Curseur

Centrer le curseur verticalement: centre le curseur de texte clignotant dans la fenêtre tant que vous ajoutez du texte à la fin du document. N'est pas recommandé pour les vieilles machines (PPC G3).
Cursor shape and behavior: (s'applique immédiatement) fournit le choix entre trois différents types de curseur: 1) le curseur «standard» fin, clignotant, 2) un curseur large, clignotant et semi-transparent et 3) un curseur fin non clignotant.
Couleur: (s'applique immédiatement) change la couleur du curseur. Si les couleurs alternatives sont activées, le curseur aura une couleur entre celle du texte et celle de l'arrière-fond. Le bouton "Par défaut" rétablit le curseur noir.

Documents

Format d'enregistrement par défaut: le format Rich Text (.rtf) fonctionne bien pour le texte formaté sans images; si vous incluez souvent des images dans vos documents, le format Rich Text avec images (.rtfd) s'y prêtera bien.

Autoenregistrer le document toutes les ... minutes: active la fonction autoenregistrement de Cocoa, qui autoenregistrera les document nouvellement ouverts à un intervalle que vous choisissez. Si Bean se plante de manière inattendue, il ouvrira le document autoenregistré la prochaine fois que vous ouvrirez le logiciel.

Nouveau modèle de document: Sélectionnez «Par défaut» pour un nouveau document typique (notez que la taille de la page par défaut fait partie des préférences d'OS X qui peuvent être modifiés dans les Préférences système > Imprimantes et fax > Format de papier par défaut. Sélectionnez «Personnalisé» afin de spécifier un modèle de document (seuls les fichiers *verrouillés* peuvent être sélectionnés). Si vous voulez verrouiller un fichier alors qu'il est ouvert par Bean, sélectionnez Ficher > Faire un modèle. Ou, sélectionnez Affichage > Afficher les informations..., puis sélectionnez «Fichier verrouillé/modèle». Un fichier peut aussi être verrouillé dans le Finder: sélectionnez le fichier dans une fenêtre du Finder, puis sélectionnez Fichier > Lire les informations, puis cochez «verrouillé».

Impression

Imprimer les en-têtes et pieds de page: (s'applique immédiatement) un en-tête et/ou pied de page est ajouté lorsque le document est imprimé sur papier ou en format .pdf. Vous verrez une prévisualisation des en-têtes et pieds de page en mode Mise en page. Notez que les en-têtes et les pieds de page peuvent disparaitre si les marges du document sont trop petites

Style: (s'applique immédiatement) modifie le contenu de l'en-tête et du pied de page imprimés. Notez que les contenus des champs Titre, Auteur-e et Sujet sont entrés par l'utilisateur dans le panneau Affichage > Afficher les propriétés… pour les documents individuels.

Ainsi, vous pouvez imprimer un en-tête de type APA:

An Analysis of Methodologies 1

...ou un en-tête de type MLA:

Kirkpatrick 1

...ou même un en-tête de style «Turabian», en ajoutant quatre espaces après le champ pour l'auteur-e.

Kirkpatrick     1

Commencer à la page: (s'applique immédiatement)
spécifie le numéro de page sur laquelle l'en-tête et le pied de page apparaissent la première fois (p. ex. si vous désirez créer une page de titre non numérotée). Le numéro de page dans l'en-tête ou le pied de page s'ajuste en fonction du réglage. Pour l'instant, vous ne pouvez pas décider par quel numéro la numérotation commencera.

Affichage

Mode mise en page: affiche le document tel qu'il apparaitra sur la page imprimée, contrairement à du texte continu. Si vous travaillez avec de longs documents (de plusieurs centaines de pages), vous aurez peut-être intérêt à désactiver l'option «Afficher le mode Mise en page» au début afin de ne pas avoir à attendre la repagination du document.

Couleurs alternatives: le document est affiché avec des couleurs alternatives (arrière-fond et texte) pour l'édition de documents. Ainsi, un texte blanc sur fond bleu peut être agréable à l'oeil. Les en-tête et pieds de page ne sont pas affichées en mode couleurs alternatives. Les couleurs de police autres que le noir restent inchangées.

Guides de marge: affiche une ligne indiquant la zone éditable d'une page qui peut être remplie de texte, c'est-à-dire la zone entre les marges. Les guides de marge ne sont pas imprimés.

Ombre de la page: ajoute une ombre agréable sous la page virtuelle, en mode Mise en page. Cependant, Bean est plus réactif si cette option est désactivée!

Comptage des mots lors de la frappe: affiche un compteur continuellement mis à jour pour le comptage des mots et caractères (ainsi que des pages, en mode Mise en page), dans la barre d'état en bas de chaque fenêtre. Si vous sélectionnez du texte, la barre d'état montrera le comptage pour le texte sélectionné (en bleu et non pas en noir).

Règle: affiche une règle au-dessus du document. Vous pouvez directement manipuler les réglages de tabulateur et de retrait pour une sélection de texte en déplaçant les marqueurs dans la règle.
Cliquez dans la règle pour créer un tabulateur de gauche. Double-cliquez sur un tabulateur (ou faites un cliquez avec la touche Option enfoncée) pour changer le type d'alignement.

Accessoires de la règle: affiche les réglages standards d'OS X dans l'espace au-dessus de la règle. Si l'option est activée, vous pouvez glisser-déposer différents types de tabulateurs depuis les tabulateurs à disposition. Vous pourrez également avoir accès aux réglages de style un peu niais d'OS X.

Barre d'outils: les éléments de la barre d'outils permettent un accès rapide et intuitif aux commandes fréquemment utilisées.

Barre de défilement horizontale: permet de défiler horizontalement dans les documents affichés avec un zoom important.

Caractères invisibles: affiche les caractères normalement invisibles (espaces, espaces insécables, tabulateurs, retours chariot, marques de paragraphes, sauts de page) afin que vous pouvez affiner la mise en page d'un document. Les grilles invisibles contenant du texte deviennent également visibles.

Couleur: spécifie la couleur des signes invisibles lorsqu'ils sont affichés.

Omettre les espaces: les espaces ne seront pas affichés lorsque la fonction «Afficher les caractères invisibles» est activée. Améliore la lisibilité.

Police

Police pour le texte RTF: la police et la taille utilisés pour les nouveaux documents. Avec un échantillon.

Police pour le texte brut: la police et la taille utilisés quand un document en texte brut est chargé (.txt, .html, etc.). Avec un échantillon.

Couleurs alternatives: la même chose que Préférences > Affichage > Couleurs alternatives. Cliquez sur les champs de couleur pour le texte et l'arrière-fond pour modifier les couleurs alternatives.

Style

Marges: détermine les réglages pour les marges de nouveaux documents. Cliquez sur «Appliquer les modifications» pour enregistrer les changements.

Interligne: détermine l'interligne pour les nouveaux documents. Un interligne de 1.2 est agréable à l'oeil.

Retrait de la première ligne: détermine le réglage pour un retrait de paragraphe automatique si vous pressez sur Entrée. Cliquez sur «Appliquer les modifications» pour enregistrer les changements.

Appliquer le style aux fichiers .txt: applique les réglages du panneau Style aux documents en texte brut ouverts par Bean. Ceci n'est pas toujours désirable. Par exemple, pour éditer du code HTML, vous ne voudrez pas du texte formaté, car il peut rendre difficile à lire le code. Mais pour un texte brut du Projet Gutenberg, en revanche, ces réglages peuvent rendre un fichier plus présentable et lisible.

Afficher en unités métriques: affiche les réglages du panneau Style en mesures métriques au lieu des mesures américaines. Ceci n'affecte pas la règle, etc. de nouveaux documents. L'unité de mesure pour la règle est automatiquement déterminée par les Préférences Système > International > Formats > Système de mesure. Vous pouvez déterminer les mesures métriques ou U.S. dans Préférences Bean > Avancées > Interface > Unités de mesure.

Appliquer les modifications: ce bouton fixe les modifiations que vous avez faits dans les réglages de texte dans ce panneau.

Fenêtre

Centrer la fenêtre initiale: les nouvelles fenêtres de document sont centrées sur l'écran, puis ajoutées en cascade vers la droite (au fur et à mesure que de nouveaux documents sont ouverts).

Taille de la fenêtre

Réglage par défaut: les nouvelles fenêtres sont ouvertes avec une taille prédéfinie.

Réglage personnel: les nouvelles fenêtres sont ouvertes avec une taille définie par l'utilisateur.

Adapter au document actuel: la taille du nouveau document sera égale à celle de la fenêtre du premier document.

Zoom

Adapter à la largeur de la page: les nouveaux documents sont mis à l'échelle de sorte que la largeur de la page imprimée soit conforme à la taille de la fenêtre. Si la fenêtre est redimensionnée par l'utilisateur, le document sera automatiquement mis à l'échelle.

sauf le texte brut: désactive «Adapter à la largeur de la page» pour les documents en texte brut. Souvent, on redimensionnera la fenêtre d'un document simplement pour éviter les lignes coupées, de sorte que la mise à l'échelle du document n'est pas nécessaire.

personnalisé: Les nouveaux documents sont mis à l'échelle, le taux étant déterminé par l'utilisateur.


Options plein écran

Marges à gauche et à droite: crègle la marge horizontale autour du texte éditable, représentée comme un pourcentage de la largeur de l'écran (maximum 33% pour chaque côté).

Cacher la règle: cache la règle en mode Plein écran et la rétablit si nécessaire en quittant le mode Plein écran.

Cacher la barre d'outils: cache la barre d'outils dans le mode plein écran et la rétablit, si nécesaire, après avoir désactivé le mode plein écran.

Couleurs alternatives: active les couleurs de texte alternatives dans le mode plein écran et les désactive, si nécessaire, après avoir désactivé le mode plein écran.

Cacher la mise en page: cache le mode Mise en page et le rétablit après avoir terminé le mode plein écran.

Avancées (document)

Permettre le changement de la couleur de l'arrière-fond: révèle une sélection de couleur dans le panneau des polices permettant de changer la couleur de fond du document. La couleur de fond du document sera imprimée (contrairement aux couleurs dans Affichage > Couleurs alternatives), mais non pas au-delà des marges du document.

Respecter le seuil d'anticrénelage / Désactiver le crénelage fin: Fait que Bean respecte le réglage des Préférences système «Apparences > Désactiver le lissage du texte pour les polices de taille de xxx et inférieure» (s'applique au prochain lancement du logiciel). Si l'utilisation des polices à l'écran peut utiliser moins de ressources système, cela désactive également le crénelage fin (c'est-à-dire l'espacement entre les caractères) pour toute taille de texte (s'applique immédiatement), de sorte qu'il n'est pas conseillé de s'en servir.

Le service 'Coller la sélection...' ajoute un séparateur: Services > Bean > Coller la sélection dans le document Bean actuel ajoute un séparateur graphique avant le texte sélectionné. Ainsi, si le service est utilisé à plusieurs reprises, les différentes sections du texte collé sont séparées graphiquement dans le document Bean. (Astuce avancée: le texte du séparateur se trouve dans defaults.plist sous prefPasteSelectionDividerString et peut être modifié par l'utilisateur.)

'Imprimer la sélection...' sépare plusieurs sélections: Un séparateur graphique est inséré entre plusieurs sélections de texte avant qu'ils soient imprimés via Fichier > Imprimer la sélection.

Espace par défaut entre les colonnes (en pts): détermine l'espace par défaut dans le contrôle Format > Colonnes (lorsqu'il y a deux ou plusieurs colonnes).

Avancées (Interface)

Unités de mesure: Bean affiche les mesures (dans la règle et les contrôles) en unités métriques ou U.S.. Le réglage automatique a pour effet que Bean utilise les unités métriques ou U.S. sur la base du réglage Préférences Système > International > Formats > Unités de mesure.

Rétablir l'emplacement du curseur lors de l'ouverture des documents: Bean rétablit le dernier emplacement du curseur clignotant quand un document est rouvert (seulement documents RTF). En général, les documents sont chargés plus rapidement si cette option est désactivée.

Afficher les numéros de page dans la barre d'état (seulement en mode Mise en page): indique dans la barre d'état les pages visibles et le nombre total de pages, en bas de la fenêtre lorsque le mode Mise en page et le comptage des mots lors de la frappe sont activés.

Afficher les numéros de page en mode Mise en page: les pages sont numérotées en mode Mise en page (en haut à gauche de chaque page). Ces numéros ne sont pas imprimés.

Afficher la règle verticale: fait ce qu'il dit. (seulement en OS X 10.5+).

Suggérer un nom de fichier lors du premier enregistrement: (s'applique immédiatement) Bean suggère un nom de fichier dans le panneau d'enregistrement, sur la base des premiers 40 signes du documents environ. (seulement enOS X 10.5+).

Avancées (Rechercher/Remplacer)

Utiliser le panneau Rechercher/Remplacer simple: révèle le panneau de recherche standard d'OS X (Rechercher > Rechercher/Remplacer), au lieu du panneau avancé de Bean. Le panneau de Bean permet l'utilisation d'expressions régulières, contrairement à celui du panneau simple du Finder.

^ et $ reconnaissent une nouvelle ligne: (s'applique immédiatement) les symboles d'expression régulière ^ et $ reconnaissent les nouvelles lignes intégrées, en plus du début et de la fin habituels d'un document. Notez que le texte trouvé n'inclut pas la nouvelle ligne, juste le texte immédiatement après ( ^ ) ou avant ( $ ).

Le point (Dot) correspond à une nouvelle ligne: (s'applique immédiatement) le symbole d'expression régulière point 'dot' ( . ) s'accorde avec la nouvelle ligne de même que l'habituel «n'importe quel signe» (sauf la nouvelle ligne). Le texte recherché inclut la nouvelle ligne.

Avancées (Mode Notes)

Déterminer la police et la couleur pour s'accorder avec le texte sélectionné: (s'applique immédiatement) établit les attributs de texte (nom de police, taille et couleur du texte, couleur de surlignage) utilisé pour le mode Notes. Ces attributs sont pris à partir du texte à l'endroit du curseur dans la fenêtre active.

Utiliser la police/taille; Utiliser la couleur du texte; Utiliser la couleur de surlignage: (s'applique immédiatement) sélectionne quels attributs vous voulez appliquer quand le mode Notes est actif.

Ajouter des parenthèses: (s'applique immédiatement) les nouvelles notes se mettent entre crochets.

Ajouter un nouveau paragraphe: (s'applique immédiatement) les nouvelles notes créent leur propre paragraphe.

Services

Rognure HTML Snippit avec styles à partir de la sélection: convertit le texte sélectionné dans les logiciels compatibles dans une rognure HTML avec CSS intégré qui s'ouvre dans une nouvelle fenêtre Bean. Les rognures HTML (texte, listes, tableaux, etc.) peuvent être collées directement dans un document HTML.

Rognure HTML sans styles à partir de la sélection: convertit le texte sélectionné dans les logiciels compatibles dans une rognure HTML sans CSS qui s'ouvre dans une nouvelle fenêtre Bean.

Nouveau document contenant la sélection: crée un nouveau document contenant le texte de la sélection (qui peut provenir d'un autre logiciel, p. ex. d'un navigateur web).

Coller ‘Lorem Ipsum’: colle un paragraphe de texte latin fictif dans n'importe quel document capable de traiter du texte.

Coller la sélection dans l'actuel document Bean: colle le texte sélectionné dans n'importe quel logiciel dans l'actuel document Bean. Afin d'ajouter un séparateur avant chaque sélection collée à l'aide de ce service, activez ce réglage: Bean Préférences > Avancées > Document > Le service 'Coller la sélection...' ajoute un séparateur.

Rétablir le texte du fichier sélectionné: rétablit le texte brut de fichiers corrompus ou obsolètes. Sélectionnez un fichier dans le Finder, puis lancez le service dans le menu Services. Un nouveau document sera créé dans Bean, qui contient le texte rétabli (le formatage, les images, etc. ne seront pas rétablis). Dans un deuxième document sera affiché le texte rejeté (en général, des suites de signes incompréhensibles). Des caractères de contrôle susceptibles de faire planter Bean seront éliminés durant le processus. Utilisez ce service pour rétablir le texte de fichiers Word 97 corrompus ou de formats de fichiers obsolètes que vous ne pouvez plus ouvrir faute de logiciel disponible pour les gérer. Notez: : Dataviz vend MacLinkPlus, qui peut faire la conversion entre de nombreux formats actuels et obsolètes.
