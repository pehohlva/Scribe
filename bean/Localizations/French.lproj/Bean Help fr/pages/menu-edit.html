Menu Édition

Couper: supprime le texte sélectionné et le garde dans le presse-papier. Notez que le texte sélectionné peut consister en plusieurs sélections non contigües. Notez également que vous pouvez faire une sélection de texte rectangulaire en pressant la touche Alt.

Copier: copie le texte sélectionné dans le presse-papier.

Coller: colle le texte copié dans le presse-papier à l'endroit du curseur (seulement pour les documents qui peuvent être modifiés).

Coller et appliquer le style actuel: colle le texte copié dans le presse-papier comme texte brut (c'est-à-dire sans formatage) à l'endroit du curseur. Le texte collé prendra la police et le style de paragraphe du texte qui le précède.

Supprimer: supprime le texte sélectionné.

Sélectionner tout: agrandit la sélection pour inclure le document entier.

Inverser la sélection: tout ce qui était sélectionné est désélectionné et vice versa.

Insérer

un saut de page/de colonne: met la suite du texte à la page suivante (ou dans la colonne suivante, en cas de plusieurs colonnes), en mode Mise en page.

un saut de paragraphe: met la suite du texte dans un nouveau paragraphe.

un saut de ligne: génère une nouvelle ligne, mais pas de nouveau paragraphe.

une espace insécable: insère une espace insécable entre des mots pour les garder ensemble (pour éviter qu'ils soient mis sur deux lignes, ce qui peut être malencontreux). 

la date (format long)

la date (format court)

l'heure

la date et l'heure

une image: insère une image au point d'insertion. Une copie de l'image est maintenue dans le document, de manière qu'il n'y a aucune référence externe pointant vers l'image originale. Vous pouvez également glisser-déposer une image directement dans le document. Si nécessaire, utilisez les réglages de l'inspecteur «Forcer la hauteur de la ligne / au maximum» afin de réduire l'espace supplémentaire autour d'une image insérée dans un paragraphe. De même, si vous placez une image entre deux sauts de paragraphe forcera l'espacement simple.

une note: il est appliqué une police et/ou couleur de contraste au texte qui est tapé immédiatement après. Cela peut être utilisé comme le crayon rouge de l'enseignant-e afin d'indiquer des notes ou des corrections. Voir Préférences Bean > Avancées > Mode Notes afin de choisir la police ou régler les options.

une ligne de signature: Insère une ligne de signature au point d'insertion.

Modifier

Texte en guillemets symétriques: remplace des guillemets droits par des guillemets symétriques (courbés), qui sont plus plaisants et typographiquement corrects.

Texte en guillemets droits: remplace des guillemets symétriques par des guillemets droits lorsque cela s'avère nécessaire (p. ex. en code source).

Taille de l'image: si le texte sélectionné contient une image, un contrôle est affiché qui permet de redimensionner l'image. Vous pouvez également double-cliquer une image pour faire apparaitre le contrôle de redimensionnement.

Supprimer

Puces (toutes/sélectionnées): supprime les marqueurs de liste et les attributs dans la sélection ou dans l'ensemble du texte si aucune sélection n'a été faite. Utile, car parfois un marqueur de liste peut être supprimé, mais l'attribut «liste» demeure, ce qui peut avoir des effets imprévisibles (marqueurs apparaissant soudain, par la suite). 

Tableaux (tous/sélectionnés): supprime les tableaux de texte dans la sélection ou dans l'ensemble du texte si aucune sélection n'a été faite. Utile, car le fait de coller du texte provenant de pages web peut avoir des effets imprévisibles en mode Mise en page (pagination inappropriée et problèmes d'affichage, bogue Apple rdar://5650712). La présence de ces tableaux de texte problématiques n'est pas toujours évidente puisque ces tableaux ne s'affichent pas sous forme de grille et ne servent qu'à positionner le texte.

Éléments attachés (tous/sélectionnés): supprime les éléments attachés (images, etc.) dans le texte sélectionné ou dans l'ensemble du texte, si aucune sélection n'a été faite.

Orthographe…

Afficher l'orthographe et la grammaire: active la vérification de l'orthographe en cochant la case «Vérifier l'orthographe» dans le panneau Orthographe et Grammaire (seulement à partir de OS X 10.5).

Parole…

Caractères spéciaux: affiche une palette pour la saisie de caractères spéciaux dans un document (lettres accentuées, etc.).
