Astuces:


Comment modifier le modèle «nouveau document»?

Sélectionnez Bean Préférences > Documents > Nouveau modèle de document: Personnalisé. Puis sélectionnez le bouton Modèle afin de localiser et de sélectionner le fichier souhaité. Important: Seuls des fichiers bloqués peuvent être sélectionnés comme modèles. Afin de créer un fichier bloqué (afin de l'utiliser comme modèle), ouvrez un document dans Bean, puis sélectionnez Fichier > Créer un modèle (ce qui bloque le fichier). Vous pouvez débloquer un fichier en le sélectionnant dans le Finder, puis en décochant l'option dans Finder > Lire les informations > verrouillé.

Comment créer un «modèle» de document réutilisable?

Créez votre document, enregistrez-le, puis sélectionnez Affichage > Afficher les informations..., puis cochez la case à côté de «Fichier verrouillé/Modèle». Rouvrez le document dans Bean. Il s'ouvrira en tant que copie sans nom du fichier d'origine. Notez: vous ne pouvez pas verrouiller un fichier qui n'a pas encore été enregistré.


Comment associer un type de fichier avec Bean afin qu'il soit toujours ouvert par Bean quand je clique sur l'icône du document?

Dans le Finder, sélectionnez un fichier du type en question. Puis, dans la barre de menu du Finder, sélectionnez Rechercher > Afficher les informations, puis Ouvrir avec: Bean. Puis cliquez le bouton 'Changer tous...' et ensuite OK.


J'utilise Convertir au format Texte / Convertir au format RTF dans TextEdit pour 'nettoyer' le texte du web de tout formatage. Y a-t-il quelque chose de semblable dans Bean?

Oui, Style > Supprimer tous les styles fait quelque chose de très similaire. Voir aussi Édition > Supprimer > Marqueurs de liste / Tableaux / Éléments attachés.


Des sélections copiés depuis internet s'affichent parfois bizarrement dans le mode Affichage > mode Mise en page. Pourquoi?

Les tableaux peuvent parfois entrainer une fausse pagination et un affichage de texte inapproprié. Les tableaux ne sont pas toujours visibles, car parfois ils n'affichent pas de grille, mais servent uniquement pour positionner le texte. Sélectionnez Affichage > Afficher les caractères invisibles pour voir les tableaux cachés. Sélectionnez Édition > Supprimer > les tableaux pour enlever les tableaux non souhaités dans une sélection.


Comment puis-je automatiquement faire un retrait pour chaque paragraphe (au lieu de presser Tab à chaque fois)?

Dans Tabulateurs par défaut dans les préférences, vous pouvez déterminer le montant pour le retrait par défaut pour les nouveaux documents. Afin d'ajuster ce réglage pour un document spécifique, sélectionnez le texte en question, ouvrez l'Inspecteur (Format > Inspecteur…), puis ajustez le retrait pour la première ligne au montant désiré.


J'en ai marre d'enlever des retours chariot non désirés dans les fichiers texte. Comment le faire facilement?

Googlez le service gratuit WordService de DEVONtechnologies, qui ajoute une commande «Format > Reformater» dans le menu OSX Services (que vous trouvez sous le menu «Bean». WordService est assez utile pour reformater le texte.


Que sont les «Guillemets intelligents»?

En activant les Guillemets symétriques dans les préférences, Bean insère automatiquement des guillemets courbés ouvrants («) ou fermants (») lorsque vous tapez du texte (à l'opposé des guillemets droits (""). Vous pouvez choisir entre différents styles de guillemets en fonction de votre langue, dans les préférences.


Afin de convertir le style de guillemets dans le texte sélectionné ou dans le document entier, sélectionnez Éditer > convertir > guillemets symétriques / guillemets droits. Notez que si vous tapez un type de guillemets par-dessus un autre, le style des guillemets alterne.


Si votre langue se sert de la touche «guillemets» comme d'une touche «morte», c'est-à-dire comme une touche que l'on utilise, en combinaison avec d'autres, pour produire des lettres spéciales, décochez l'option Préférences > Général > Guillemets symétriques. Sous Leopard, vous pouvez vous servir des guillemets symétriques gérées par Cocoa. Faites un clic de droite ou un Ctrl-clic dans le texte du document, puis sélectionnez dans le menu déroulant l'option Substitutions > Guillemets symétriques afin d'activer les guillemets symétriques gérées par Cocoa.


Pourquoi je vois parfois des guillemets symétriques même si j'ai désactivé l'option?

Certaines polices sont suffisamment avancées pour mettre des guillemets courbés à l'écran ou sur la page imprimée même si ceux dans le texte sont tous droits.


Pourquoi les italiques ne fonctionnent pas avec (p. ex.) Lucida Grande?

Pour améliorer la qualité du rendu, OS X utilise différentes polices pour le gras, les italiques, etc. (avant, les ordinateurs ont simulé ces variantes en ajoutant du gras ou de l'inclinaison). Mais toute option n'existe pas pour chaque police. Parfois, vous avez beaucoup plus de choix. Regardez p. ex. Helvetica Neue. Notez que vous pouvez modifier le style de police du texte sélectionné facilement au moyen d'un menu déroulant dans l'Inspecteur de Bean.


Que devrais-je savoir par rapport au travail avec des images dans un document?

Vous ne pouvez non seulement utiliser Édition > Insérer > Image pour l'importation d'une image, mais vous pouvez aussi en glisser une depuis le Finder ou la copier-coller depuis un logiciel de traitement d'image. Afin de redimensionner une image, double-cliquer sur l'image à redimensionner afin de faire apparaitre le réglage de redimensionnement. Alternativement, sélectionnez du texte de manière à ce que la première image dans la sélection est celle qui doit être redimensionnée. Puis sélectionnez Édition > Changer > Taille de l'image afin de faire apparaitre le réglage de redimensionnement.


Comment faire fonctionner les références aux images dans le code HTML?



Suivez ces étapes:


1. Ouvrez le document HTML dans Bean (glissez l'icône sur celle de Bean).
2. Replacez le code de référence pour les images dans le code HTML (précédé par img src=) par les noms des fichiers d'images que vous placerez dans le même dossier que le fichier HTML (ou ainsi: images/logo.jpg s'il y a un dossier «images» qui contient les images).
3. Enregistrez-le, puis double-cliquez le fichier HTML afin de l'ouvrir dans Safari, puis vérifiez son apparence.


Comment faire un rectangle autour d'un texte?

Officiellement, vous ne pouvez pas le faire. Mais il est simple d'utiliser la fonction «tableau» pour le simuler, comme je viens de le faire dans le cadre ci-dessus.



Suivez ces étapes:


1. Sélectionnez le texte que vous voulez mettre dans un cadre

2. Dans le menu: Format > Tableau

3. Faites un «tableau» avec une ligne et une colonne

4. Ajustez la bordure et l'arrière-fond de la cellule.


NOTE IMPORTANTE: La fonction Tableau de texte peut être très inconstante. Vous pouvez forcer la récréation correcte d'un tableau en sélectionnant une ligne de texte en dessus du début du tableau, puis en pressant Return et ensuite Delete. Malheureusement, tel est le fonctionnement du Cococa Frameworks pour les objets texte, je ne peux donc pas le réparer.


Comment sélectionner du texte comportant un lien? Il s'ouvre toujours dans Safari!

Positionnez le curseur juste à côté du texte avec lien et utilisez [Maj] + flèche pour faire la sélection.


Quelle est la différence entre l'autoenregistrement et la sauvegarde?

• Afin d'activer l'autoenregistrement pour tous les documents, sélectionnez Préférences Bean > Documents > Autoenregistrer documents. Cela autoenregistre les documents à l'aide de la fonction autoenregistrement native de Cocoa, à un intervalle que vous avez spécifié. Si Bean se plante inopinément, le logiciel rouvrira la dernière version autoenregistrée du document, de sorte que vous ne perdrez pas vos modifications. Cela est disponible pour tous les formats de documents. La fonction ne porte que sur les documents ouverts après son activation.

• Afin d'activer la sauvegarde munie d'une date pour un document individuel, sélectionnez Affichage > Afficher les informations..., puis cochez «Sauvegarde toutes les … minutes», en ajustant l'intervalle désiré. La sauvegarde munie d'une date étant activée, une copie de la version actuelle du document est enregistrée à un intervalle déterminé. La sauvegarde créera un fichier dans le même dossier que le document d'origine, mais avec le suffixe «(sauvegarde)». La sauvegarde n'écrase jamais le fichier d'origine et ne revient jamais à la version sauvegardée après un plantage (ce que fait l'autoenregistrement). Il y a une sauvegarde munie d'une date si le document a été modifié depuis la dernière sauvegarde (que le document ait été enregistré ou non). Contrairement aux documents autoenregistrés, la copie de sauvegarde n'est pas supprimée après la fermeture ou l'enregistrement du document. La sauvegarde munie d'une date n'est disponible que pour les documents RTF.

• «Sauvegarde à la fermeture» crée automatiquement une copie du document enregistré lorsqu'il est fermé. Pour activer «Sauvegarde à la fermeture», sélectionnez Affichage > Afficher les informations…, puis «Sauvegarde à la fermeture». Une copie de votre fichier sera faite dans le même dossier que l'original, mais la date actuelle sera suffixée au nom du fichier (de même qu'un numéro de série si le fichier est sauvegardé plus d'une fois par jour). Vous pouvez aussi créer une copie de sauvegarde munie d'une date n'importe quand en sélectionnant Fichier > Archiver depuis le menu. Sauvegarde à la fermeture n'est disponible que pour les documents RTF et doit être activé pour chaque document individuellement. Sauvegarde à la fermeture est conçu pour le contrôle des versions, la documentation des modifications ou l'archivage


Une copie de sauvegarde munie d'une date peut être créée à tout moment en sélectionnant Fichier > Sauvegarde, depuis le menu.


Comment accélérer l'accès au grand dictionnaire?
Après avoir sélectionné Rechercher > Définir mot, choisissez Cacher (cmd-H) afin de cacher le Dictionnaire, tout en le laissant tourner en arrière-fond.


Comment imprimer comme PDF?

Sélectionnez Fichier > Imprimer, puis cliquez sur le bouton 'PDF', ensuite 'Enregistrer en format PDF...' dans le menu contextuel. Alternativement, sélectionnez Ficher > Exporter > en format PDF.

Comment désinstaller Bean?

1. Glissez l'icône du logiciel Bean depuis le dossier où il se trouve (probablement Applications) dans l'icône de la corbeille dans le Dock. Vous devez peut-être vous authentifier pour le faire. (Si vous ne trouvez pas Bean.app, cherchez-le avec Spotlight. Si vous ne le trouvez toujours pas, il se pourrait qu'il se trouve dans CopyPaste Pro.bundle si vous utilisez Copypaste Pro.)
2. Vous pouvez aussi mettre à la poubelle le fichier suivant (préférences de Bean):
/Utilisateurs/[Vous]/Library/Preferences/com.SevenYearsDreaming.Bean.plist
