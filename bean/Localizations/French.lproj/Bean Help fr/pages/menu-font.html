Menu Polices

Polices…: affiche le panneau pour les polices standard d'OS X, qui vous donne également accès aux réglages de typographie avancés (ligatures, lettres alternatives, etc.).

Couleurs…: affiche le sélecteur de couleurs standard d'OS X. Si vous sélectionnez une couleur dans le sélecteur, elle s'utilisera pour le texte sélectionné dans un document. Pressez Option+Maj en sélectionnant une couleur depuis le sélecteur pour modifier la couleur de surlignage du texte (c'est-à-dire, la couleur de fond du texte sélectionné).

Une note à propos des styles de police: OS X utilise des polices différentes pour le gras, les italiques, etc. afin d'augmenter la qualité (les ordinateurs plus vieux les ont simulés en ajoutant du gras ou de l'inclinaison). Mais chaque option n'existe pas pour toute police. Parfois, vous avez bien plus d'options. Prenez comme exemple l'Helvetica Neue. Notez que vous pouvez modifier le style de la police du texte sélectionné facilement à partir du menu contextuel dans l'inspecteur de Bean.

Mode Notes: S'il est activé, une combinaison de police et/ou de couleur contrastive s'applique au texte qui est tapé n'importe où dans le document. Cela peut s'utiliser comme le crayon rouge de l'enseignant-e afin de marquer les notes ou les corrections nécessaires. Voir Préférences Bean > Avancées > Mode Notes pour déterminer les options liées à la police ou à la couleur. Resélectionner l'option afin de la désactiver.
