Raccourcis clavier


Dans le menu, vous trouverez les raccourcis clavier pour beaucoup de fonctions spécifiques de Bean.

En plus, voici quelques raccourcis clavier standards pour l'édition de textes dans Mac OS X. Certains de ces raccourcis ne sont pas évidents, surtout pour qui vient d'un environnement Windows PC.


⌃⌘D  ...  dictionnaire contextuel (dictionnaire intégré dans OS X)
[esc]  ...  suggestions pour compléter le mot (intégré dans OS X)
⌘↑  ...  placer le cursor au début du document
⌘↓  ...  placer le cursor à la fin du document / forcer la repagination du document
⌥←  ...  placer le curseur au mot à gauche
⌥→  ...  placer le curseur au mot à droite
⌃↑  ...  voir la page en haut
⌃↓  ...  voir la page en bas
fn↑  ...  défiler à la page supérieure
fn↓  ...  défiler à la page inférieure
fn⌥↑  ...  placer le curseur une page plus haut
fn⌥↓  ...  placer le curseur une page plus bas
⇧↑  ...  sélectionner la ligne supérieure
⇧↓  ...  sélectionner la ligne inférieure
fn⇧↑  ...  sélectionner la page supérieure
fn⇧↓  ...  sélectionner la page inférieure
⇧⌘↑  ...  sélectionner jusqu'au début du document
⇧⌘↓  ...  sélectionner jusqu'à la fin du document
⌥⌫  ...  supprimer le mot à gauche
fn⌥⌫  ...  supprimer le mot à droite
⌥[  ...  forcer des guillemets droits (si les 'guillemets symétriques' sont sélectionnée)
⌥]  ...  forcer un guillemet simple (si les 'guillemets symétriques' sont sélectionnés)

⌃[retour chariot]  ...  insérer un retour chariot (seulement dans Leopard)

⌥[barre d'espace]  ...  insérer une espace insécable

[bouton de défilement]⌥  ...  défilement très lent (seulement dans Leopard)

[ctrl]t  ...  intervertit les deux lettres les plus proches du point d'insertion


Dans les listes:


⌥[tab]  ...  créer une nouvelle liste

[tab]  ...  mettre en retrait un élément de la liste

⇧[tab]  ...  enlever le retrait d'un élément de la liste

⌃⌥[tab]  ...  insérer un tabulateur dans un élément d'une liste

⌥⌘→  ...  mettre en retrait un élément de la liste (seulement dans Bean)

⌥⌘←  ...  enlever le retrait d'un élément de la liste (seulement dans Bean)

NOTEZ: [retour] sur une ligne vide enlève le retrait de l'élément nouveau suivant


Panneau Rechercher:


⌥[tab]  ...  insère un tabulateur dans la case de recherche

⌥[retour]  ...  insère un retour charion dans la case de recherche

[option]  ...  Remplacer tout/sélectionner tout (dans le document entier) devient Remplacer tout/sélectionner tout (seulement dans la sélection)


Panneau Couleur:


⇧[option]  ...  sélectionne la couleur de surlignement (couleur de fond) pour le texte sélectinné au lieu de sélectionner la couleur de la police (seulement dans Bean)

Légende pour les symboles:


⌃ contrôle (ctrl)

⌘ commande

⌥ option (ou alt)

fn fonction

⇧ majeur (shift)

⌫ delete


Symbols Legend:


⌃ control key

⌘ command  key

⌥ option (or alt)

fn function key

⇧ shift key

⌫ delete key
